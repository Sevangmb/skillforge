### **Démarrer l'Application**
```bash
npm run dev
```
→ Accès : http://localhost:3000

### **Navigation**

1. **Faire le défi quotidien** : Cliquez "Commencer le défi"
2. **Générer nouveau parcours** : Bouton "Nouveau" dans les parcours
3. **Voir la progression** : Statistiques en temps réel
4. **Suivre les streaks** : Système de séries quotidiennes
